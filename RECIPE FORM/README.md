# RECIPE PAGE #
For this assignment, I was tasked to create a professional and appetizing recipe webpage by applying my foundational HTML and CSS skills to build a complete, two-file project consisting of an index.html and a style.css file. I was to structure the content using semantic HTML tags to organize sections like a hero image, ingredient lists, step-by-step intructions, and nutritional facts, and then style the page with CSS to demonstrate mastery of concepts such as the box model, typography, and modern layouts with Flexbox and Grid, ultimately producing a clean, responsive, and easy-to-read final design.
